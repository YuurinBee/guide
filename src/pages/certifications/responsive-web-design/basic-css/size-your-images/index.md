For sizing your images, you need to first create your class in your style brackets and name it. 


An example is:
```
<style>
  .fixed-image {
    width: 500px;
    height: auto;
  }
  ```
  
  You can then add the class to your image, like below:
  ```
  <img class="fixed-image" src="http://www.example.com/picture"/>
  ```
